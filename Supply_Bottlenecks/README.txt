# Supply Bottlenecks and Equity Market Forecasting

## Description
This project aims to investigate how weather volatility, satellite night light intensity, and media sentiment impact equity market forecasting. The goal is to integrate these diverse data sources to improve predictive accuracy.

## Current Status
At the idea and planning stage. Preliminary assignment completed; no code or draft yet.

## Planned Methods
- Time series econometrics (e.g., VAR, EGARCH)
- Integration of climate, satellite, and sentiment data for forecasting

## Next Steps
- Develop data collection pipelines
- Draft initial literature review and methodology
